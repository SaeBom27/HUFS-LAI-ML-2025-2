# 3차 과제 제안서 (202402960 신새봄)

## 스페인어 동사 분류 모델

### 필요성
- 스페인어는 동사 변화가 매우 다양하고 복잡한 언어이다.
- 같은 동사라도 법(Mood), 시제(Tense), 인칭(Person), 수(Number)에 따라 100가지 이상의 완전히 다른 형태로 바뀐다.
- 때문에 학습 중 문장 속에서 변형된 동사를 보았을 때 "이 동사가 어떤 법의 어떤 시제이지?"하고 헷갈리는 일이 자주 있다.
- 그러나 기존의 사전이나 번역기는 동사의 문법적 역할보다는 의미에 초점을 두어 문법적 형태 정보를 제공하지 않는다.
- 따라서 스페인어 학습 시 유용하게 활용할 수 있는 스페인어 동사 분류 모델을 만들고자 한다.

### 솔루션
- 변형된 스페인어 동사를 입력하면, 그 동사의 문법적 형태 정보를 자동 분석해주는 도구를 만든다.
- 예) [habláremos] -> [접속법, 미래, 1인칭, 복수]
- 주요 기능
  - 동사의 철자만으로 법(Mood), 시제(Tense), 인칭(Person), 수(Number)를 자동 분류
  - 규칙/불규칙 동사 모두 지원
- 스페인어 읽기 수업이나 문법 설명이 부족한 교재를 공부할 때 사용할 것이다.

### ML 활용 계획
- 데이터 수집 및 분석
  - [unimorph spanish] (https://github.com/unimorph/spa/blob/master/spa.derivations)
  - 위의 데이터를 활용하여 규칙 기반 활용형 생성기를 통해 [동사 - 라벨] 의 데이터셋을 만든다.
  - 예) {"verb" : "habláremos", "mood" : "subjunctive", "tense" : "future", "person" : 1pl}
  - 클래스별 분포를 분석하여 만약 데이터가 부족하거나 불균형하다면 보완할 다른 데이터를 수집하여 추가한다.
- 모델 학습 및 평가
  - Char 기반의 Multi-Head LSTM 분류 모델을 학습시킬 예정이다.
    - 동사를 문자 단위로 임베딩하여 LSTM으로 특징을 추출한다.
    - 각각의 head에서 법,시제,인칭,수를 동시에 예측하는 구조로 만든다.
  - 평가는 Accuracy, Macro-F1등을 통해 성능을 확인할 예정이다.
